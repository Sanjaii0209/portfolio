Customer Churn Prediction

Overview

This project aims to predict customer churn for a telecom company using machine learning algorithms. The model helps to identify customers likely to stop service, enabling proactive retention strategies.

Explanation (Making)

The data was cleaned and processed using Pandas. Exploratory Data Analysis (EDA) identified key churn drivers. Various classifiers such as Logistic Regression and Random Forest were evaluated. Hyperparameter tuning improved the model's performance.

Image

The image above shows the confusion matrix of the best-performing model, demonstrating accuracy and balance between false positives and negatives.

Conclusion

The model achieved 85% accuracy in predicting churn. This predictive tool provides valuable insights to help the company retain customers and optimize marketing strategies.
